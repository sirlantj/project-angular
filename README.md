## Prerequisites

* Node.js 14.16.0+
* Angular Cli 13+

## Build / Run

```
git clone https://github.com/sirlantj/example-angular
npm install
ng serve
```

## Configuration 

- You need to change the var API_URL inside the file data.service.ts for you API url
